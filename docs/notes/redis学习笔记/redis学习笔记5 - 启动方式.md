## 五 、Redis 启动

### 5.1 Redis 以守护进程服务端进行启动

```bash
# 使用该命令进行启动: 【./bin/redis-server ./redis.conf 】
./bin/redis-server ./redis.conf
```

### 5.2 Redis 客户端进行启动

```bash
# Redis 客户端启动命令: ./redis-cli
./redis-cli
```
