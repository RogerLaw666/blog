## 概述

- fortune：输出一段格言警句、名著节选或者小笑话等。
- cowsay：将输出的文字内容包含在由 ASCII 字符组成的动物形象的气泡内。
- lolcat：将苍白的输出文字变成绚丽的彩虹。

## 安装

```bash
sudo apt install fortune cowsay lolcat
```

## 使用

```bash
fortune|cowsay|lolcat
```
