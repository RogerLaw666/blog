## 概述

ES6，全称 ECMAScript 6.0，是 JavaScript 的下一个版本标准，2015.06 发版。

ES6 主要是为了解决 ES5 的先天不足，为其添加了许多新的语法特性，ES6 是 ECMAScript 标准十余年来变动最大的一个版本。
